# Modified Default TADA68 layout

This layout replicates the default factory layout of the TADA68 with:
 + Home/End/Ins shifted down one key
 + New layer for:
    + Media Keys
    + "Vim" key binding
    + Emergency Magic SysRq
 + NumPad

```
(Base Layer) Default Layer
,----------------------------------------------------------------.
|Esc | 1|  2|  3|  4|  5|  6|  7|  8|  9|  0|  -|  =|Backsp |~ ` |
|----------------------------------------------------------------|
|Tab  |  Q|  W|  E|  R|  T|  Y|  U|  I|  O|  P|  [|  ]|  \  |Del |
|----------------------------------------------------------------|
|CAPS/FN|  A|  S|  D|  F|  G|  H|  J|  K|  L|  ;|  '|Return |PgUp|
|----------------------------------------------------------------|
|Shift   |  Z|  X|  C|  V|  B|  N|  M|  ,|  .|  /|Shift | Up|PgDn|
|----------------------------------------------------------------|
|Ctrl|Win |Alt |        Space          |Alt| FN|Ctrl|Lef|Dow|Rig |
`----------------------------------------------------------------'
```
